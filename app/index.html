Run This Project

This project uses the Grunt task runner. Grunt has to be installed with node and npm. See the README.md file in the root folder of the projet for more information on how to setup the project.

  • Use grunt serve to run launch a Web server and open that index page on your browser.

Test This Project

In order to test this project just add your assertions to tests/test.js.

  • Use grunt testBrowser to test on your browser.
  • Use grunt testCI for Continuous Integration to test on PhantomJS, the headless browser.
